---
title: Now available – Google Tag Manager first-party mode in the Cloudflare dash
description: Cloudflare's integration with Google Tag Manager lets you deploy a Google tag using your CDN's first-party infrastructure
date: 2025-04-07T16:56:49.182
---

Cloudflare's support for Google Tag Manager's first-party mode is now available in the dashboard in a closed beta. When your site is behind Cloudflare, this integration enables you to load your Google tags directly from Cloudflare's CDN infrastructure upon page load, giving you greater control and accuracy over analytics measurements. 

You can get started first by following instructions on [Google's developer docs](http://developers.google.com/tag-platform/tag-manager/first-party/setup-guide.md). If you'd like to edit your configuration, you can do so by heading to [Tag Management > Google tag](https://dash.cloudflare.com/?to=/:account/tag-management/googletagfirstparty) in the Cloudflare dashboard.
